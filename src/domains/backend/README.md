以 Java 生态为主线，覆盖语言基础、Spring 全家桶、持久层框架、数据库原理与调优、主流中间件，以及系统架构设计与实战项目。从单机应用到分布式系统，兼顾理论深度与工程落地。

- [**数据库**](./databases/) — MySQL 原理与调优、Redis 缓存、分布式事务
- [**中间件**](./middleware/) — 消息队列（Kafka / RocketMQ）、注册中心、网关
- [**架构设计**](./architecture/) — 系统设计案例、软件架构风格、软考系统架构师备考
- [**项目实战**](./projects/) — 完整项目经验与架构复盘
