放瓶子

题面

小美初始位于 (a,b)(a, b)(a,b) 位置，一维平面上有 nnn 个瓶子，每个瓶子的位置为 (xi,yi)(x_i, y_i)(xi​,yi​)。小美每次可以向上、下、左、右移动一格，每次移动的代价为 111，小美需要移动到第一个瓶子的位置上，然后拿起瓶子把它放到 (c,d)(c, d)(c,d) 位置，每次最多只能拿一个瓶子，请问最少需要多少代价才能把所有瓶子都放到 (c,d)(c, d)(c,d) 位置上。

输入描述:
第一行四个整数 a,b,c,d(−109≤a,b,c,d≤109)a, b, c, d(-10^9 \leq a, b, c, d \leq 10^9)a,b,c,d(−109≤a,b,c,d≤109)，表示小美初始位置和瓶子需要放置的位置。
接下来一行一个整数 n(1≤n≤105)n(1 \leq n \leq 10^5)n(1≤n≤105)，表示瓶子的数量。
接下来 nnn 行，每行两个整数 xi,yi(−109≤xi,yi≤109)x_i, y_i(-10^9 \leq x_i, y_i \leq 10^9)xi​,yi​(−109≤xi​,yi​≤109)，表示第 iii 个瓶子的位置。

输出描述:
输出一个整数，表示最少需要多少代价。

贪心。

我们需要计算小美从初始位置到每个瓶子的距离，然后从每个瓶子到目标位置的距离，最后将所有瓶子移动到目标位置的总距离。关键是要找出一个最优的瓶子访问顺序，以最小化总移动距离。

最有访问顺序：找一个 离 (a,b)(a, b)(a,b) 点距离 + (c,d)(c, d)(c,d) 点距离最大 的点作为第一个点，其他点的移动距离都是 到 (c,d)(c, d)(c,d) 点距离*2

  1. 计算每个瓶子到位置 (a,b)(a, b)(a,b) 和 (c,d)(c, d)(c,d) 的距离之和，取最大。

  2. 计算剩余瓶子到位置 (c,d)(c, d)(c,d) 的距离。

  3. 累加所有移动的距离即可。

最大乘积

题面

小美有三个数字 a,b,ca,b,ca,b,c，他每次操作可以选择一个数字将其 +1+1+1，最多可以操作 kkk 次，小美想知道 a×b×ca \times b \times ca×b×c 的最大值是多少？

由于这个数字可能很大，因此你需要输出答案对 109+710^9 + 7109+7 取模后的结果。

输入描述：
第一行四个正整数 a,b,c,k(1≤a,b,c,k≤1018)a,b,c,k(1 \leq a,b,c,k \leq 10^{18})a,b,c,k(1≤a,b,c,k≤1018) 表示数字和操作次数。

输出描述：
输出一个整数表示答案。

贪心。

应该将所有的操作次数用在最小的数字上，直到它等于或超过第二小的数字，然后再考虑第二小的数字，以此类推。这样可以最大化三个数字的乘积。

  1. 将三个数字排序，使 a≤b≤ca ≤ b ≤ ca≤b≤c。

  2. 计算将 aaa 增加到 bbb 所需的操作次数，如果 kkk 足够，就执行这些操作。

  3. 如果还有剩余操作次数，尝试将 aaa 和 bbb 同时增加到 ccc。

  4. 如果还有剩余操作次数，平均分配到 aaa、bbb、ccc 上。

  5. 计算最终的 a×b×ca × b × ca×b×c，并对 109+710^9 + 7109+7 取模。

def max_product(a, b, c, k):
    MOD = 10**9 + 7
    
    # 步骤1：排序确保 a <= b <= c
    nums = sorted([a, b, c])
    a, b, c = nums
    
    # 步骤2：将 a 增加到 b
    diff = b - a
    if k >= diff:
        a = b
        k -= diff
    else:
        a += k
        return (a * b * c) % MOD
    
    # 步骤3：将 a 和 b 增加到 c
    diff = (c - a) * 2
    if k >= diff:
        a = c
        b = c
        k -= diff
    else:
        increase = k // 2
        a += increase
        b += increase
        if k % 2 == 1:
            a += 1
        return (a * b * c) % MOD
    
    # 步骤4：平均分配剩余的操作次数
    increase = k // 3
    a += increase
    b += increase
    c += increase
    
    # 处理剩余的1或2次操作
    if k % 3 == 1:
        a += 1
    elif k % 3 == 2:
        a += 1
        b += 1
    
    # 步骤5：计算最终结果并取模
    return (a * b * c) % MOD

# 读取输入
a, b, c, k = map(int, input().split())

# 计算并输出结果
result = max_product(a, b, c, k)
print(result)

牛牛商店

题面

牛牛商店里只卖两种商品，"牛可乐"和"牛马克"。

现在有 nnn 个人来到商店购物，第 iii 个人有喜好区间 [li,ri][l_i, r_i][li​,ri​] 和购买目标商品，他只看货架上位于区间里的商品，并从中挑选 kik_iki​ 个保质期最长的牛可乐或者牛马克走（如果有多个商品保质期相同，他会拿走区间中靠前的那个）。你能告诉牛牛每个人买走的商品编号吗？

输入描述：
第一行输入两个整数 nnn 和 mmm (1≤n,m≤105)(1 \leq n, m \leq 10^5)(1≤n,m≤105) 代表来牛牛商店购物的人数和商品数量。
第二行输入 mmm 个整数 a1,a2…ama_1, a_2 \ldots a_ma1​,a2​…am​ (1≤ai<109)(1 \leq a_i < 10^9)(1≤ai​<109) 代表商品的保质期。
第三行输入 mmm 个整数 b1,b2…,bmb_1, b_2 \ldots, b_mb1​,b2​…,bm​ (0≤bi≤1)(0 \leq b_i \leq 1)(0≤bi​≤1) 代表商品的种类，其中，bi=0b_i = 0bi​=0 代表"牛可乐"，bi=1b_i = 1bi​=1 代表"牛马克"。
此后 nnn 行，第 iii 行输入四个整数 li,ri,til_i, r_i, t_ili​,ri​,ti​ 和 kik_iki​ (1≤li≤ri≤m;0≤ti≤1;1≤ki≤m)(1 \leq l_i \leq r_i \leq m; 0 \leq t_i \leq 1; 1 \leq k_i \leq m)(1≤li​≤ri​≤m;0≤ti​≤1;1≤ki​≤m) 代表第 iii个人的喜好区间、购买商品种类和购买数量。其中，ti=0t_i = 0ti​=0 代表他想要买"牛可乐"，ti=1t_i = 1ti​=1 代表他想要买"牛马克"。

输出描述：
输出 nnn 行，第 iii 行输出至多 kik_iki​ 个整数，代表第 iii 个人购买的商品编号（如果有多个商品保质期相同，输出编号较小的那个）。你需要按照从小到大的顺序依次输出；如果没有买到足够的商品，使用一个 −1-1−1 替代。

核心是高效地在指定区间内找出保质期最长的特定类型商品。考虑到数据规模 (n,m≤105)(n,m ≤ 10^5)(n,m≤105)，我们需要一个能够在 O(logn)O(log n)O(logn) 时间内查询区间最大值的数据结构。即线段树。

  1. 构建两个线段树，分别用于存储牛可乐和牛马茶的保质期信息。

  2. 在线段树中，每个节点存储该区间内的最大保质期和对应的最小商品编号。

  3. 对于每个顾客的查询：

    • 在对应类型的线段树上查询指定区间 [li,ri][l_i, r_i][li​,ri​] 内的最大保质期商品。
    • 如果找到符合条件的商品，记录其编号，并在线段树中将该商品的保质期设为 −1-1−1（表示已售出）。
    • 重复上述过程 kik_iki​ 次或直到无法找到更多符合条件的商品。
